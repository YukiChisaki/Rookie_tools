

## 项目使用的skills

| Skill | 用途 | 说明 |
|-------|------|------|
| `naive-ui` | Naive UI 组件开发 | Vue 3 组件库使用指南，包含 33 条规则涵盖组件、主题、表单、数据展示等 |
| `vue` | Vue 3 开发 | Composition API、script setup、响应式系统、生命周期等最佳实践 |
| `vite` | Vite 配置与构建 | Vite 配置、插件开发、SSR、构建优化等 |
| `ui-ux-pro-max` | UI/UX 设计 | 50+ 设计风格、配色方案、字体组合、动画效果，支持 Tailwind CSS |
| `frontend-design` | 前端界面设计 | 创建精美的 Web 界面、组件、页面布局 |
