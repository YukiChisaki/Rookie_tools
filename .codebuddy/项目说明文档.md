# 项目说明文档

## 项目定位

面向 Stable Diffusion WebUI 与 ComfyUI 用户的本地离线 Prompt 工具箱。

## 核心功能

1. **图片 Meta 解析**
   - 解析 PNG 图片中的 ComfyUI workflow 与 SD WebUI 参数
   - 自动提取正向 / 负向提示词、模型、采样器等关键信息
   - 支持批量导入与拖拽上传
   - 解析结果可一键导入提示词管理

2. **标签选择器**
   - 七类标签：画师、风格、质量、构图、光影、角色、环境
   - 搜索支持名称与别名匹配
   - 点击标签自动加入选择，支持权重调整（0.1 - 2.0）
   - 一键复制已选标签为提示词格式

3. **提示词管理**
   - 保存、编辑、删除提示词记录
   - 正向 / 负向提示词分开管理
   - 支持从图片解析结果导入
   - 可复制、导出为文本

4. **画师与画师串管理**
   - 画师数据展示与收藏
   - 画师风格快速复制
   - 画师串（组合多个画师风格）的创建与管理
   - 点击画师 / 画师串直接插入提示词

## 技术栈

- 框架：Vue 3 + TypeScript
- 构建：Vite
- 状态管理：Pinia
- 样式：Tailwind CSS
- 存储：IndexedDB（本地离线）
- 图标：Lucide Vue Next

## 项目结构

```
rookie_tools/
├── src/
│   ├── modules/          # 四大功能模块
│   ├── components/       # 通用组件
│   ├── services/         # 数据服务层
│   ├── stores/           # Pinia 状态管理
│   ├── composables/      # Vue 组合式函数
│   ├── types/            # TypeScript 类型定义
│   ├── utils/            # 工具函数
│   └── styles/           # 全局样式
├── data/lists/           # 标签数据文件
└── public/               # 静态资源
```

## 数据存储

所有数据存储于浏览器 IndexedDB，支持完全离线访问：
- `tags`：标签数据
- `prompts`：提示词记录
- `parsedImages`：解析的图片数据
- `artists`：画师数据
- `artistChains`：画师串数据

## 标签数据源

标签数据从 `data/lists/` 目录下的 txt 文件加载：
- artists.txt、style.txt、quality.txt、composition.txt、lighting.txt、character.txt、environment.txt

## 快速开始

- 安装依赖：`npm install`
- 启动开发服务器：`npm run dev`
- 构建生产版本：`npm run build`
- 预览生产版本：`npm run preview`
