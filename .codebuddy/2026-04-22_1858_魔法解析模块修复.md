# 魔法解析模块修复记录

**日期**: 2026-04-22  
**时间**: 18:58

## 修改内容

### 1. 新增文件
- `src/services/spellParser.ts` - 图片元数据解析服务
- `src/stores/spell.ts` - 魔法解析状态管理
- `src/modules/spell-parser/` - 魔法解析模块组件

### 2. 修改文件
- `src/types/index.ts` - 添加魔法解析相关类型定义
  - `GeneratorType`: 生成器类型（comfyui/stable-diffusion/novelai/unknown）
  - `ImageParameters`: 图片生成参数接口
  - `ParsedImageData`: 解析后的图片数据结构
  - 扩展 `ModuleType` 添加 'spell'
  - 更新 `DB_VERSION` 到 2，添加 `PARSED_IMAGES` store

### 3. 修复问题
1. **Stable Diffusion 图片解析失败**
   - 问题：`parameters` 字段未被识别，导致被标记为 unknown
   - 解决：在 `detectGenerator` 中添加对 `parameters` 字段的检测

2. **ComfyUI 图片解析失败**
   - 问题：`workflow` 和 `prompt` 字段是以 `{ value: "..." }` 对象格式存储
   - 解决：新增 `extractFieldValue` 辅助函数，支持提取字符串或对象的 value 属性

3. **正负提示词显示颠倒**
   - 问题：按节点顺序分配提示词，但 ComfyUI 节点顺序不保证正向在前
   - 解决：通过 KSampler 节点的 `inputs.positive` 和 `inputs.negative` 连接关系来判断

4. **新增详细日志输出**
   - 在控制台输出完整的解析过程日志，便于调试
   - 包括：元数据标签、生成器类型检测、节点解析、参数提取等
