# Rookie Tools - 需求与设计文档 (RPD)

## 1. 项目概述

### 1.1 项目背景
Rookie Tools 是一款面向 Stable Diffusion WebUI 和 ComfyUI 用户的本地离线 Prompt 工具箱，旨在帮助 AI 绘画爱好者更高效地管理提示词、解析图片参数、整理画师风格资源。

### 1.2 目标用户
- Stable Diffusion WebUI 用户
- ComfyUI 用户
- NovelAI 用户
- AI 绘画爱好者与研究者

### 1.3 核心价值
- **离线使用**：所有数据存储在本地浏览器，保护隐私
- **参数解析**：一键解析 AI 生成图片的完整参数
- **资源管理**：系统化整理标签、提示词、画师资源
- **高效工作流**：快速复制、组合、复用提示词

| TS-005 | 一键复制已选标签为提示词格式 | 高 |
| TS-006 | 使用频率统计与排序 | 低 |

#### 模块三：提示词管理

| PM-004 | 提示词列表搜索 | 中 |
| PM-005 | 支持复制正向/负向/全部提示词 | 高 |
| PM-006 | 区分手动创建和解析导入的提示词 | 低 |

| NF-003 | 历史记录限制 | 解析历史最多保存 50 条 |
| NF-004 | 深色/浅色主题切换 | 支持双主题，自动保存偏好 |

**生成器检测逻辑：**
```
检测流程：
1. 检查 workflow/prompt 字段 → 判定为 ComfyUI
2. 检查 Software/Comment 包含 NovelAI → 判定为 NovelAI
3. 检查 parameters 字段 → 判定为 Stable Diffusion
4. 检查 sd-metadata 字段 → 判定为 Stable Diffusion
5. 以上都不匹配 → 标记为 Unknown
```

## 4. 接口设计

### 4.1 图片解析服务接口

```typescript
// 解析图片文件
function parseImage(file: File): Promise<ParseResult>

interface ParseResult {
  success: boolean;
  data?: ParsedImageData;
  error?: string;
}

// 格式化生成器名称
function formatGeneratorName(generator: GeneratorType): string

// 格式化参数显示
function formatParameter(key: string, value: unknown): string
```

### 4.2 数据库服务接口

```typescript
// 初始化数据库
function init(): Promise<void>

// 基础 CRUD 操作
function get<T>(storeName: string, id: string): Promise<T | undefined>
function getAll<T>(storeName: string): Promise<T[]>
```

### 4.3 图片压缩工具接口

```typescript
// 压缩图片文件
function compressImage(file: File, options?: CompressOptions): Promise<CompressedImages>

interface CompressedImages {
  thumbnailData: string;  // Base64 缩略图 (120x120)
  previewData: string;    // Base64 预览图 (原比例压缩)
}

interface CompressOptions {
  thumbnailSize?: number;     // 默认 120
  thumbnailQuality?: number;  // 默认 0.8
  previewQuality?: number;    // 默认 0.85
  type?: string;              // 默认 'image/jpeg'
}
```

## 5. 技术选型

| 类别 | 技术 | 版本 | 选型理由 |

## 6. 风险评估

| 风险点 | 影响 | 缓解措施 |
|-------|-----|---------|
| IndexedDB 容量限制 | 中 | 限制历史记录数量（50条），压缩图片存储 |
| 浏览器兼容性 | 低 | 使用标准 API，测试主流浏览器 |
| 大图片解析性能 | 中 | 异步解析，加载状态反馈 |
| 元数据格式变化 | 低 | 适配主流生成器常见格式 |
